# Learning Material

A collection of interactive learning apps built with modern web technologies.

## Projects

### Animal Quiz

An interactive quiz game that tests your knowledge of animals, habitats, and wildlife. Built with React and Vite.

**Features:**
- 4 difficulty levels: Easy, Medium, Hard, and Extra Hard
- ~89 unique questions across categories like general animals, fun facts, and wildlife & habitats
- Countdown timer that adjusts per difficulty (5:00 down to 1:30)
- Instant answer feedback with visual highlights
- Score tracking with grade-based results
- Persistent leaderboard (top 20 scores via localStorage)
- Nature-themed UI

**Tech stack:** React 19, Vite 7, ESLint 9

### Getting Started

```bash
cd animal-quiz
npm install
npm run dev
```

**Available scripts:**
| Command | Description |
|---|---|
| `npm run dev` | Start the development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview the production build |
| `npm run lint` | Run ESLint |
